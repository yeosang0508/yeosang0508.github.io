Spring Security

  • - 이를 통해 Spring Security의 보안 설정과 JWT 기반 인증 체계를 학습하고 실무에 적용하는 경험을 쌓고 있습니다.

JSP

  • - JSP를 사용하여 웹 애플리케이션의 View 레이어를 구현하며 서버에서 전달된 데이터를 기반으로 동적 웹페이지를 생성했습니다.
  • - 서블릿과 JSP 간의 데이터 전달을 통해 사용자 요청을 처리하고, 데이터베이스에서 조회된 정보를 웹페이지에 표시하는 작업을 수행했습니다.
  • - MVC 패턴을 적용하여 역할을 분리하고, JSP를 사용자 인터페이스(View)로 활용하면서 서블릿에서 비즈니스 로직과 데이터 처리를 담당하도록 설계했습니다.

Thymeleaf

  • - Spring Boot와 Thymeleaf를 연동하여 서버사이드 렌더링 기반의 웹 애플리케이션을 개발했습니다.
  • - Thymeleaf 템플릿 엔진을 활용하여 HTML5와 데이터를 결합하고, 정적인 콘텐츠를 효과적으로 표시하는 작업을 수행했습니다.
  • - 컨트롤러에서 전달된 데이터를 템플릿에 삽입하여, 서버에서 처리된 결과를 사용자 인터페이스(View)에 전달했습니다.
  • - 템플릿의 재사용성을 높이기 위해 Thymeleaf의 `layout` 기능을 활용하여 공통 레이아웃을 설계하고, 유지보수성을 강화했습니다.

React

  • - 현재 진행 중인 프로젝트에서 React를 사용하여 컴포넌트 기반의 사용자 인터페이스를 구현하는 작업을 진행 중입니다.
  • - Redux를 활용하여 전역 상태 관리를 설계하고 있으며, 컴포넌트 간 데이터 흐름을 효율적으로 처리할 수 있도록 구현 중입니다.
  • - React Router를 적용하여 페이지 간의 동적 라우팅을 구성하고, 사용자 경험을 개선하기 위한 네비케이션 기능을 구현 중입니다.
  • - 이러한 작업을 통해 React와 Redux를 활용한 상태 관리, 라우팅 설계, 그리고 사용자 인터페이스 개발 경험을 쌓아가고 있습니다.
